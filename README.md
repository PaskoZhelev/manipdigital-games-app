# ManipDigital Games

ManipDigital Games is a collection of small, single-player logic puzzle games built with React and Vite. Each game focuses on concise, thoughtful puzzle mechanics and runs entirely in the browser.

[Logic Gems Game](https://paskozhelev.github.io/manipdigital-games-app/logic-gems/)

[Wild Lies Game](https://paskozhelev.github.io/manipdigital-games-app/wild-lies/)

Included games (high level):

- Symbiomes — place animals on a grid to satisfy environment constraints.
- Neon Protocol — pattern and logic puzzles with daily challenges.
- Logic Gems — deduction puzzles combining color and shape clues.
- Wild Lies — deduction puzzles to find the culprit in a case.

Games provide daily levels and store progress locally in the browser. Levels are generated automatically by repository workflows.
